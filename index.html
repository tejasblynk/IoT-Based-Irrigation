üìñ Project Abstract

This intelligent irrigation system utilizes an ESP8266 NodeMCU to monitor soil moisture via a capacitive sensor. The system operates in two primary modes: Automatic Mode, which uses hysteresis logic to activate a water pump when moisture levels drop below 680 ADC, and Manual Mode, which allows for global remote control via the Sinric Pro IoT platform. Local feedback is provided via a flicker-free 16x2 LCD display.

üíª Core Firmware Logic

void autoWateringControl(int rawValue) {
  if (!autoModeEnabled) return;

  // DRY Condition (Hysteresis Logic)
  if (rawValue > 680 && !autoPumpRunning) {
    digitalWrite(RELAY_PIN, HIGH);
    autoPumpRunning = true;
    pumpSwitch.sendPowerStateEvent(true);
  }
  // WET Condition
  else if (rawValue <= 680 && autoPumpRunning) {
    digitalWrite(RELAY_PIN, LOW);
    autoPumpRunning = false;
    pumpSwitch.sendPowerStateEvent(false);
  }
}
