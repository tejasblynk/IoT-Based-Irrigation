üìã How It Works

  1. Moisture Monitoring: Capacitive sensor continuously reads soil moisture level
  2. Data Processing: ESP8266 converts analog readings to percentage (0-100%)
  3. LCD Display: Shows real-time moisture, pump status, and mode
  4. Auto Watering: When moisture drops below 68%, pump turns ON automatically
  5. Pump Control: Relay switches pump ON/OFF based on moisture threshold
  6. IoT Updates: Sinric Pro receives updates every 5 seconds
  7. Remote Control: Users can control via app or voice commands
  8. Manual Override: App control automatically switches to manual mode

üéØ System Specifications

Moisture Thresholds (Calibrated for ESP8266):

  • VERY_DRY: 830 (ADC value) ‚Üí Soil needs immediate watering
  • NEITHER_DRY_OR_WET: 680 ‚Üí Threshold for auto mode
  • VERY_WET: 330 ‚Üí Soil is adequately watered

Update Intervals:

  • LCD Update: Every 1 second (only changed values to prevent flickering)
  • Sinric Update: Every 5 seconds
  • Auto Watering Check: Every 2 seconds

3. Upload Code

  • Open Arduino IDE
  • Select Board: NodeMCU 1.0 (ESP-12E Module)
  • Select correct COM Port
  • Click Upload

4. Calibrate Sensor

1. Open Serial Monitor (115200 baud)
2. Note ADC value when sensor is in DRY soil
3. Note ADC value when sensor is in WET soil
4. Update calibration values in code
5. Re-upload code

üì± Sinric Pro Setup

  1. Create account at sinric.pro
  2. Create 3 devices:
    • Soil Sensor - Type: Contact Sensor
    • Pump - Type: Switch
    • Auto Mode - Type: Switch
  3. Copy Device IDs and paste in code
  4. Link Sinric Pro with Alexa/Google Home app
  5. Test voice commands: "Alexa, turn on the pump"

üéÆ Usage Guide

Auto Mode (Default):

  • System monitors soil moisture automatically
  • When moisture < 30%, pump turns ON
  • When moisture > 40% (hysteresis), pump turns OFF
  • LCD shows "AUTO" mode

Manual Mode:

  • Control pump via Sinric app
  • Use voice commands
  • Auto mode automatically disables
  • LCD shows "MANU" mode

Re-enable Auto Mode:

  • Toggle "Auto Mode" switch in Sinric app
  • Or restart the ESP8266

üêõ Troubleshooting

LCD Not Displaying:

  • Check I2C address (try 0x3F if 0x27 doesn't work)
  • Verify SDA/SCL connections
  • Adjust LCD contrast potentiometer

WiFi Not Connecting:

  • Check SSID and password
  • Ensure 2.4GHz WiFi (ESP8266 doesn't support 5GHz)
  • Move closer to router

Pump Not Working:

  • Verify relay connections
  • Check if relay LED lights up
  • Ensure common ground between ESP and power supply
  • Test pump with direct power connection
